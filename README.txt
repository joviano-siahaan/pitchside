1. Download XAMPP (https://www.apachefriends.org/xampp-files/8.0.7/xampp-windows-x64-8.0.7-0-VS16-installer.exe)
2. Open XAMPP, start APACHE & MYSQL
3. Open PHPMYADMIN (localhost/phpmyadmin)
4. Click New
5. Database Name = laravel
6. Import > Select laravel.sql from Google Drive
7. Unzip pitchside.zip to directory/xampp/htdocs
For Example: C:\xampp\htdocs\skripsi
8. Open Command Prompt
9. Change directory to xampp
For Example: cd C:\xampp\htdocs\pitchside
10. Run "php artisan serve"
11. Open browser > http://127.0.0.1:8000/
